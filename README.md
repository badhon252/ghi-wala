# ঘি ওয়ালা — Premium Landing Page

Production-ready Next.js + TypeScript + Tailwind CSS v4 landing page.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Before production

- Replace `https://ghiwala.example.com` in `app/layout.tsx` with the real domain.
- Add real product price/variant data if available.
- Connect the order form to a CRM/API if you need server-side lead storage. Currently it opens WhatsApp with a pre-filled order.
- Add final legal policy pages and business address.
